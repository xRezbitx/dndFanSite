What is D&D?

Dungeons and Dragons is a Tabletop Roleplaying game. This means that you play the game as a character that you create! Unless you're running the game as a DM.

What is a 'DM'?

It stands for 'Dungeon Master' - which is a fun title for the person who runs the game! They control the game world and everything in it, except for your created characters.
